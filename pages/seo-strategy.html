Immediate Actions (Week 1-2)

Foundation setup for search engine visibility and performance tracking:

1. Google Search Console Setup for rid.academy

Step-by-Step Instructions:
  1. Go to https://search.google.com/search-console
  2. Sign in with your Google account (admin@ekwamarketing.com recommended)
  3. Click "Add Property" and select "URL prefix"
  4. Enter: https://rid.academy
  5. Verify domain ownership via one of these methods:
    • DNS record verification (RECOMMENDED): Add TXT record to rid.academy DNS
    • HTML file upload to domain root
    • Meta tag in homepage HTML head
    • Google Analytics account verification
    • Google Tag Manager account verification
  6. After verification, submit sitemap.xml
  7. Monitor "Coverage" tab for any errors/warnings
Pro Tip: Use DNS verification for fastest and most reliable verification method. Once verified, Google will begin crawling and indexing rid.academy.

2. Bing Webmaster Tools Setup

Step-by-Step Instructions:
  1. Go to https://www.bing.com/webmasters/
  2. Sign in with Microsoft account
  3. Click "Add site"
  4. Enter: https://rid.academy
  5. Verify ownership via:
    • Meta tag (same one used for Google Search Console works here)
    • XML file upload
    • CNAME record verification
  6. Connect Google Analytics (if using UA, otherwise use GA4)
  7. Submit sitemap.xml: https://rid.academy/sitemap.xml
  8. Monitor "Index" tab for crawl health
Bing Benefits: Bing powers approximately 6-9% of search traffic in the US. While smaller than Google, Bing users are often high-intent. DuckDuckGo also indexes from Bing, expanding your reach.

3. Google Analytics 4 Setup for rid.academy

Step-by-Step Instructions:
  1. Go to https://analytics.google.com
  2. Click "Create" for new account or select existing Ekwa Marketing account
  3. If creating new: Name = "RID Academy", select timezone and currency
  4. Create a new "Property" named "rid.academy"
  5. Select "Web" as the platform
  6. Enter website URL: https://rid.academy
  7. Copy the "Measurement ID" (format: G-XXXXXXXXXX)
  8. Install Google Analytics tag on all rid.academy pages:
    • Option A: Use Google Tag Manager (GTM) container
    • Option B: Add GA4 tag directly to website header
  9. Create conversion goals/events:
    • Newsletter signup
    • PDF download
    • Consultation request
    • Podcast episode click
    • Book purchase clicks
  10. Set up organic traffic segment to track SEO performance

4. Submit Sitemap to Search Engines

Google Search Console Sitemap Submission:
  1. In Google Search Console, navigate to "Sitemaps" (left menu)
  2. Enter sitemap URL: https://rid.academy/sitemap.xml
  3. Click "Submit"
  4. If you have separate sitemaps for different content types, submit each:
    • https://rid.academy/sitemap-articles.xml
    • https://rid.academy/sitemap-guides.xml
    • https://rid.academy/sitemap-hubs.xml
  5. Monitor submission status and coverage
Bing Webmaster Tools Sitemap Submission:
  1. In Bing Webmaster Tools, go to "Sitemaps" (left menu)
  2. Click "Submit sitemap"
  3. Enter: https://rid.academy/sitemap.xml
  4. Click "Submit"
